
# Golang CSV Viewer

Visualize CSV data in Golang

## Run

### `official` example

```bash
cd official
go run .

# in another terminal window
open http://localhost:8089
```

### `bars` example

```bash
cd bars
go run main.go
```

### `kline` example

```bash
cd kline
go run main.go
```
